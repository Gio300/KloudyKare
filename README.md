# KloudyKare

Static frontend for Kloudy Kare, deployed via GitHub Pages.

## URLs

- **GitHub Pages (test):** `https://YOUR_USERNAME.github.io/KloudyKare/`
- **Production:** `https://kloudykare.com` (when DNS configured)

## Apps

- `/` - User login
- `/admin/` - Admin dashboard
- `/eligibility/` - Nevada Medicaid eligibility
- `/approvals/` - Admin approvals

## Production API

Set `window.KLOUDY_API_BASE = "https://api.kloudykare.com"` in `js/config.js` when deploying to production.
